O que são Clients

Um cliente é qualquer dispositivo ou aplicação que solicita um recurso de um servidor na internet. No contexto de HTML, o cliente é responsável por processar o conteúdo recebido (como arquivos HTML, CSS e JavaScript) e exibi-lo ao usuário.

Funcionamento

O cliente envia uma requisição HTTP ao servidor (ex.: ao digitar um URL ou clicar em um link); O servidor responde com os arquivos necessários (ex.: um arquivo HTML e seus recursos); O cliente processa e renderiza a página para o usuário.

Exemplos de Clients

  • Navegadores Web

    Também conhecidos como Google Chrome, Mozilla Firefox, Safari, Microsoft Edge.

  • Aplicativos Móveis

    Aplicativos instalados no computador ou em dispositivos móveis que consomem dados ou apresentam páginas web.
