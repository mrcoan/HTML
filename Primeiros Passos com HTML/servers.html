O que são Servers?

Um servidor é um computador ou software que fornece recursos, serviços ou dados para outros computadores (os clientes) na rede. Servidores web são especificamente projetados para armazenar e entregar conteúdos como arquivos HTML, CSS, JavaScript, imagens, vídeos, e outros tipos de dados.

Exemplo de funcionamento

  1. O cliente digita www.exemplo.com no navegador;
  2. O navegador envia uma requisição para o servidor onde o site está hospedado;
  3. O servidor responde com o arquivo HTML e outros recursos (CSS, JS, imagens).
  4. O navegador renderiza e exibe a página ao usuário.
