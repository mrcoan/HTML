História dos computadores

Os computadores transformaram todos os aspectos da vida moderna, desde o trabalho e a educação até o entretenimento e a pesquisa científica, evoluindo de máquinas gigantes para dispositivos portáteis atuais. Cada era foi marcada por mudanças tecnológicas que revolucionaram o modo como as pessoas trabalham, se comunicam e interagem com a tecnologia.

O que são Clients?

Em HTML, clients geralmente se referem aos dispositivos ou navegadores que solicitam, recebem e interpretam páginas web do servidor. Esses "clientes" são a parte front-end da comunicação no modelo cliente-servidor, comum na web.

O que são Servers?

Em HTML e no contexto da web, servers (ou servidores) são os computadores ou sistemas que armazenam, processam e enviam os arquivos necessários para que os clientes (como navegadores) exibam páginas web. Eles desempenham um papel fundamental no modelo cliente-servidor da Internet

Como é uma estrutura básica de um HTML?

A estrutura básica de um arquivo HTML segue um padrão bem definido, que inclui elementos essenciais para criar uma página web.

  • !DOCTYPE html

    Declara que o documento é escrito em HTML5. É necessário para que os navegadores interpretem o arquivo corretamente.

  • html e /html

    A tag html envolve todo o conteúdo da página.
    O atributo lang="pt-BR" informa ao navegador que o idioma principal do documento é português do Brasil

  • head e /head

    Contém informações sobre a página (metadados). Essas informações não aparecem diretamente no conteúdo exibido ao usuário.
    meta charset="UTF-8" define a codificação de caracteres como UTF-8, permitindo o uso de caracteres especiais (acentos, por exemplo)
    meta name="viewport" content="width=device-width, initial-scale=1.0" Torna a página responsiva, ajustando a visualização em dispositivos móveis
    title Define o título que aparece na aba do navegador

  • body e /body

    Contém o conteúdo visível da página. Tudo o que está dentro dessas tags será exibido no navegador.
    h1 para defirnir o Título principal da página;
    p para digitar parágrafos de texto.

Diferença entre listas ordenadas e não ordenadas

Em HTML, listas ordenadas e não ordenadas são formas de apresentar itens de forma estruturada. A diferença entre elas está no tipo de marcação e na maneira como os itens são exibidos no navegador

Lista ordenada

Uma lista onde os itens possuem uma ordem explícita, geralmente numerada (1, 2, 3...). Normalmente utilizada para sequências, instruções passo a passo ou qualquer situação onde a ordem dos itens seja importante.

  1. Primeiro Item
  2. Segundo Item
  3. Terceiro Item

Lista não ordenada

Uma lista onde os itens não possuem ordem explícita, sendo apresentados com marcadores (geralmente círculos ou discos). Geralmente utilizada em grupos de itens onde a ordem não importa, como listas de características, tarefas ou opções.
